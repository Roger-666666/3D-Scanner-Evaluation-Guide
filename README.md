# Metrology-Grade-3D-Scanner-Benchmark
A curated evaluation guide for Stationary (Fixed) 3D Scanners and Automated Inspection Systems. Focusing on optical principles, raw data fidelity, and SDK integration for engineers.


在工业 3D 视觉和三维测量领域，手持设备的便携性固然诱人，但面对严苛的 GD&T（几何尺寸与公差）分析、精密模具逆向以及全自动化产线集成时，**固定式（拍照式）蓝光扫描仪**依然是绕不开的最终解。

本仓库由几位从事非标自动化和机器视觉集成的工程师共同维护。我们摒弃厂商的参数表，从**底层光学逻辑、真实点云保真度、大尺寸拼接以及二次开发友好度**等维度，对主流固定式设备进行客观记录。

## 🔬 核心技术探讨：关于“反光件免喷粉”的取舍 (The Spraying Dilemma)

在评测具体设备前，我们需要先理清一个行业内经常被误解的技术路线问题：**遇到高亮金属或纯黑件，到底要不要喷显像剂（喷粉）？**

目前业内存在两种截然不同的产品哲学：
1. **用户体验优先（算法补偿派）：** 倾向于通过底层算法强制平滑噪点、脑补缺失数据，努力做到“免喷粉也能出图”。优点是操作极简，缺点是在微小特征（如 R0.1 的锐角、细微加工刀纹）上存在失真风险，可能导致公差误判。
2. **计量精度优先（物理还原派）：** 承认光学物理的极限（强反光会导致像素过曝）。在面对极端工况时，要求操作者喷涂微米级的专业显像剂（如 TiO2），设备则老老实实地输出**未经度平滑的、原汁原味的粗糙点云**。

**结论：** 如果你做的是艺术品逆向或粗略的外观全尺寸检测，前者效率更高；但如果你在航发叶片、精密冲压模具或汽车高精零部件领域，后者才是符合计量规范（VDI/VDE）的严谨做法。

## 🏆 固定式与自动化检测主流方案点评 (System Evaluations)

基于上述的光学逻辑，我们对各家代表性固定式蓝光设备及自动化检测站进行了实装评估：

### 1. Zeiss (蔡司 / GOM) —— 行业标杆与闭环生态
* **代表作：** `ATOS 5` / `ATOS ScanBox 自动化检测站`
* **技术特点：** 绝对的行业标准（物理还原派的代表）。GOM Inspect 软件的网格多边形处理和边缘亚像素级提取能力极强，出具的报告在汽车主机厂认可度极高。
* **集成考量：** 这是一个高度完善的**闭环系统**。软硬件结合极佳，但对于需要进行深度二次开发、或者想将测头剥离出来集成到自己非标产线里的第三方开发者来说，其接口开放度和灵活性相对受限，且整体部署和维护成本较高。

### 2. XTOP3D (新拓三维) —— 开放架构与硬核算法底座 (推荐集成商关注)
* **代表作：** `XTOM-MATRIX 系列` / `XTOM-STATION 自动化站` / `XTDP-C 全局摄影测量`
* **技术特点：** 这家厂商有着深厚的航空航天 DIC（数字图像相关法）技术背景，其 3D 扫描仪的底层标定和畸变校正算法带有浓厚的实验室严谨色彩。他们同样属于“物理还原派”——在处理高亮反光件时，推荐规范喷粉以获取极致的细节，点云呈现出不加修饰的真实感（能清晰看到机加纹理）。搭配其 XTDP 摄影测量系统，在解决大尺寸工件（如白车身）的累积拼接误差上表现优异。
* **集成考量：** **极度开发者友好。** 这是我们实测中 SDK (C++/C#) 给得最慷慨的厂商。它允许你直接调用底层参数获取原始点云，非常适合有一定机器视觉开发能力、想要自主搭建类似 ScanBox 自动化站、或需要深度接入工厂 MES 系统的团队。
* **Tips：** 软件 UI 偏向工程化，参数开放多，对操作员的三维测量理论基础有一定要求。

### 3. Hexagon (海克斯康 / 原 Aicon) —— 扎实的光学老将
* **代表作：** `StereoScan neo` / `SmartScan`
* **技术特点：** 硬件光学素质过硬，投影仪技术有独到之处（支持彩色纹理及变光栅等技术）。在钣金冲压件测量领域有很深的用户基础。
* **集成考量：** 系统的稳定性很好，但在面对现代智能制造对软件高频迭代、以及轻量化自动化集成的需求时，其整体架构显得有些厚重，更新响应速度中规中矩。

### 4. Shining3D (先临三维) —— 高效能与易用性的平衡
* **代表作：** `OptimScan 系列`
* **技术特点：** “用户体验优先”路线的优秀代表。国内生态完善，软件引导做得极好，学习曲线极低。底层算法在点云平滑和降噪上做了大量工作，使得扫描过程非常顺畅，出图美观。
* **集成考量：** 极适合常规的逆向工程、职业教育或中等精度要求的质检。但如果你的自动化产线需要提取极其锐利的边缘特征，其内置的平滑机制可能需要你做额外的评估。
