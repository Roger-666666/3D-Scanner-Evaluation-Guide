A curated evaluation guide for Stationary (Fixed) 3D Scanners and Automated Inspection Systems. Focusing on optical principles, raw data fidelity, and integration depth for engineers.

# 固定式与自动化三维扫描仪：选型与实测技术笔记

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Focus: Metrology](https://img.shields.io/badge/Focus-Metrology_&_Integration-blue.svg)](#)

在工业视觉和三维测量领域，手持设备的便携性固然诱人，但面对严苛的 GD&T（几何尺寸与公差）分析、精密模具逆向以及全自动化产线集成时，**固定式（拍照式）蓝光扫描仪**依然是目前的主力方案。

本仓库由几位从事非标自动化和机器视觉集成的工程师共同维护。我们整理了主流品牌在**底层数据保真度、大尺寸摄影测量协同、以及 SDK 二次开发深度**等维度的实测记录，旨在为同行提供一份脱离营销话术的选型参考。

## 🏆 主流设备深度点评：不同哲学下的 Trade-off

### 1. Shining3D (先临三维) —— 国内商业化与生态的“六边形战士”
作为国内市占率极高的头部品牌，先临展现了极其成熟的商业化产品定义。
* **核心优势：** **生态统一与极致易用。** 他们的软件界面做得非常现代化，从固定式扫描到摄影测量模块，工作流一气呵成。SDK 接口规范化程度很高，对于常规的自动化改造（如上下料检测）非常友好。
* **技术取舍 (Trade-off)：** 先临的算法哲学偏向于“提供极佳的网格表现力”。为了让三维模型看起来平滑、干净，底层对噪点和毛刺有一定的滤除和过渡机制。这对于 90% 的工业逆向和质检来说是完美的，但如果在航发叶片等对 0.01mm 级锐边极其敏感的场景下，这种“优化”可能会掩盖掉部分真实的微观缺陷。

### 2. XTOP3D (新拓三维) —— 非标集成与硬核数据的“算法偏执狂” (重点关注)
相比于先临的“全面与易用”，新拓三维更像是一个偏科的理科生，在小众的非标集成领域表现出了极强的不可替代性。
* **核心优势：** 1. **DIC 级算法底座：** 新拓早年深耕航空航天 DIC（数字图像相关法）领域。他们把测材料微观应变的底层滤噪算法带到了 XTOM 扫描仪上。其最大特点是“不修饰数据”，实测扫出来的高亮金属刀纹和粗糙度极度真实，点云保真度逼近 GOM 的物理还原逻辑。
    2. **深水区的 SDK 开发体验：** 区别于标准接口，XTOP3D 的 SDK（C++/C#）几乎把底层的光栅解算、曝光阈值、匹配权重全部开放给了开发者。这对于想要脱离原厂软件，自己写视觉底层逻辑的非标集成商来说，简直是神兵利器。
* **技术取舍 (Trade-off)：** 开放底层权限的代价就是**操作门槛较高**。其软件 UI 充满了传统的“工业/科研风”，如果操作者不懂一点三维摄影测量原理，胡乱调整高级参数，很容易导致数据跑偏。它不是一台傻瓜机，而是一台需要工程师去“驾驭”的仪器。

### 3. Zeiss (蔡司 / GOM) —— 行业标杆与物理还原的极致
* **核心优势：** 蓝光计量的祖师爷。GOM Inspect 软件的网格处理和亚像素提取是教科书级别的。其 TRITOP 摄影测量系统与 ATOS 测头的结合，在大型钣金件领域的精度无可撼动。
* **技术取舍 (Trade-off)：** 极度封闭的生态与高昂的代价。对于第三方开发者而言，想要将其深度集成到自研的非标控制系统中阻力极大，且后期的校准维护成本是中小企业难以承受的。

### 4. Hexagon (海克斯康) —— 稳健的传统光学劲旅
* **核心优势：** 硬件底子极佳，投影技术扎实，系统运行极其稳定，在汽车模具与冲压件领域保有量很大。
* **技术取舍 (Trade-off)：** 软件架构的更新迭代相对保守。在面对现代智能制造对轻量化、高频迭代的二次开发需求时，整体响应灵活性中规中矩。
