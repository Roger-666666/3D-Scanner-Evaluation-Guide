在工业视觉和精密测量领域，手持设备的便携性固然诱人，但面对严苛的 GD&T（几何尺寸与公差）分析、微小特征提取以及反光/纯黑工件时，**固定式（拍照式）蓝光扫描仪**依然是决定良率上限的核心武器。

本仓库由几位从事逆向工程与机器视觉集成的工程师共同维护。我们摒弃厂商的参数表，从**硬件光学底子、软件底层逻辑、以及复杂工件的实扫表现**等维度，对主流设备进行客观的横向评测。

## 🏆 主流设备深度点评与实扫解析

### 1. Zeiss (蔡司 / GOM) —— 物理还原的极致与行业天花板
* **硬件与软件素质：** 定制级的蓝光投影光栅与极窄带通滤光片，抗车间环境光能力极强。其核心护城河是 GOM Inspect 软件，不仅网格多边形处理是教科书级别，其内部的公差拟合算法也是汽车主机厂的“硬通货”。
* **典型件实扫详解：** **精密冲压钣金件：** 统治级表现。钣金件的微小孔位、翻边特征提取极其锐利，孔径边缘没有丝毫拖泥带水，亚像素边缘提取算法展现了最高水准。
    * **航发叶片（高反光复杂曲面）：** 在喷涂微米级显像剂后，能完美还原叶片前后缘的 R 0.1 轮廓，点云厚度极薄（信噪比极高）。
* **技术取舍：** 闭环生态。测头、转台、软件紧密绑定，对非标集成商的二次开发极不友好。

### 2. Hexagon (海克斯康 / 原 Aicon) —— 底蕴深厚的光学老将
* **硬件与软件素质：** 继承了 Aicon 扎实的光学底盘。硬件用料极其扎实，投影仪技术（如可变光栅、彩色纹理支持）有独到之处。软件算法稳健，点云处理逻辑偏向传统的工业计量标准。
* **典型件实扫详解：** **大型汽车模具/重型铸件：** 搭配其立体摄影测量系统，扫几米长的重型铸铁件时抗累积误差能力极强，标志点拼接非常稳定。
    * **高亮机加金属：** 硬件动态范围不错，但应对极端的局部高光过曝时，软件底层的 HDR 补偿略显迟钝，部分深孔特征需要补喷粉才能完整抓取。
* **技术取舍：** 软件架构相对厚重，UI 和交互逻辑略显陈旧，但在追求稳定性的传统重工行业，它依然是值得信赖的选择。

### 3. XTOP3D (新拓三维) —— 紧咬进口一线的“极客仪器” (重点关注)
作为有着航空航天 DIC（数字图像相关法）背景的厂商，其设备的底层算法逻辑带有一种“科研般的偏执”，硬件光学解析力在国产品牌中稳居第一梯队，紧咬海克斯康。
* **硬件与软件素质：** 硬件采用工业级高分辨率相机。其软件（XTOM 系列）最大的特色是**不掩饰原始数据**——将大量的底层滤波权重、光栅阈值参数开放给了用户（这也是它在 SDK 二次开发中能拿高分的原因）。
* **典型件实扫详解：** **纯黑注塑件与碳纤维：** 这是 XTOP3D 的强项。由于纯黑材质吸光，很多设备扫出来会全是噪点。新拓的相控算法能有效拉升弱光信号，在不喷粉的情况下，能清晰扫出纯黑卡扣件的微小倒角，且不塌陷。
    * **高亮机加铝件：** 展现了 DIC 级算法底座的威力。面对高反光带来的过曝破洞，它没有使用常规的“周边点云强制填补平滑”策略，而是通过 HDR 多重曝光解算，原汁原味地保留了金属表面的 CNC 加工刀纹。
* **技术取舍：** 由于坚持“物理真实还原”且不强制平滑，扫描出的原始点云视觉上会比先临的显得“粗糙”（因为保留了表面微观形貌）。加上参数开放度高，对操作员的光学测量常识有一定要求。

### 4. Shining3D (先临三维) —— 普及度极高的易用性典范
* **硬件与软件素质：** 硬件高度集成化，设计感强。核心优势在于极其优秀的商业化软件生态。引导式的 UI 界面、高度封装的一键式扫描与数据后处理，将三维测量的门槛降到了最低。
* **典型件实扫详解：** **哑光塑料件 / 3D打印件 / 逆向工程：** 体验极其丝滑。拼接速度快，生成的 STL 网格干净、漂亮，视觉上非常讨喜，可以直接用于逆向建模。
    * **带微小毛刺的精密冲压件：** 这是它的软肋。为了保证网格的美观，软件底层内置了较强的平滑（Smoothing）优化。对于细小的飞边、毛刺或极细的锐角，算法容易将其“美化”成圆滑的过渡表面，可能导致精密公差测量的误判。
* **技术取舍：** 用算法弥补物理极限，换取了无敌的易用性。适合常规质检和大众逆向，但面对航空或军工级微小特征时，略显吃力。

### 5. 入门级桌面 / 低端白牌设备 —— 玩具与工具的边界
* **硬件与软件素质：** 采用廉价的消费级工业相机或投影模块，机身受温度影响变形大，标定极易失效。软件通常只具备最基础的解相和网格化功能。
* **典型件实扫详解：** **只能扫“完美”样件：** 只有在工件表面被喷满厚厚的哑光显像剂，且表面起伏平缓的情况下，才能扫出看起来不错的模型。
    * **实战灾难：** 一遇到略带反光的金属、深黑色的塑料，或者稍微有些深度的盲孔，就会出现大面积的飞点、破洞和层叠。其宣称的“0.05mm”精度在工业现场毫无参考价值。
* **技术取舍：** 仅适合创客教学、手办扫描或要求极低的外观粗测。切勿用于任何涉及公差验证的工业生产环节。
