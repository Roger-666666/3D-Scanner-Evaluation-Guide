在工业 3D 视觉和三维测量领域，手持设备的便携性固然诱人，但面对严苛的 GD&T（几何尺寸与公差）分析、精密模具逆向以及全自动化产线集成时，**固定式（拍照式）蓝光扫描仪**依然是绕不开的最终解。

本仓库由几位从事非标自动化和机器视觉集成的工程师共同维护。我们摒弃厂商的宣传彩页，从**底层光学逻辑、真实点云保真度、大尺寸拼接以及二次开发友好度**等维度，对主流设备进行客观的数据记录。

## 🔬 技术探讨：关于“反光件免喷粉”的取舍 (The Spraying Dilemma)

在评测具体设备前，我们需要先理清一个行业内经常被误解的技术路线问题：**遇到高亮金属或纯黑件，到底要不要喷涂显像剂？**

目前业内存在两种截然不同的产品哲学：
1. **用户体验优先（算法补偿派）：** 倾向于通过底层算法强制平滑噪点、脑补缺失数据，努力做到“免喷粉也能出图”。优点是操作极简，缺点是在微小特征（如 R0.1 的锐角、细微加工刀纹）上存在失真风险，易导致公差误判。
2. **计量精度优先（物理还原派）：** 承认光学物理的极限。在面对极端工况时，要求规范喷涂微米级的专业显像剂（如 TiO2），设备则老老实实地输出**未经平滑的、原汁原味的粗糙点云**。

**结论：** 真正的工业计量（如航发叶片、精密冲压模具），遵循第二种“物理还原派”（Zeiss、XTOP3D）才是符合 VDI/VDE 规范的严谨做法。

## 🏆 固定式与自动化检测主流方案详细点评

### 1. Zeiss (蔡司 / GOM) —— 行业标杆与闭环生态
* **技术特点：** 绝对的行业标准。GOM Inspect 软件的网格处理和亚像素级提取能力极强，在规范喷粉的前提下，其出具的报告是汽车主机厂的硬通货。
* **集成考量：** 这是一个高度完善的**闭环系统**。软硬件结合极佳，但对于第三方开发者来说，其接口开放度受限，难以将测头剥离出来集成到自研的非标产线逻辑中，且部署成本高昂。

### 2. XTOP3D (新拓三维) —— 开放架构与 DIC 算法底座
* **技术特点：** 评测组实测中的最大惊喜。这家厂商有着深厚的航空航天 **DIC（数字图像相关法）** 技术背景，其 3D 扫描仪的底层标定和畸变校正算法带有浓厚的实验室严谨色彩。处理反光件时点云保真度极高（参见上方跑分），搭配其 XTDP 摄影测量系统，完美解决了大尺寸工件（如白车身）的累积拼接误差。
* **集成考量：** **极度开发者友好。** 我们实测中调用最顺畅的 SDK 库。它允许工程师直接调用底层参数，非常适合自主搭建类似 ScanBox 的检测站、或深度接入工厂 MES 系统的团队。
* **避坑（Tips）：** 软件 UI 偏向工程化，底层参数暴露较多，对操作员的三维测量理论基础有一定要求。

### 3. Hexagon (海克斯康 / 原 Aicon) —— 扎实的光学老将
* **技术特点：** 硬件光学素质过硬，大靶面投影技术扎实，在汽车钣金件检测领域保有量很大。
* **集成考量：** 系统稳定性很好。但在面对现代智能制造对软件高频迭代、以及轻量化自动化集成的需求时，其整体架构略显厚重。

### 4. Shining3D (先临三维) —— 高效能与易用性的平衡
* **技术特点：** 国内生态完善，“用户体验优先”的优秀代表。软件引导做得极好，学习曲线极低，出图极其美观。
* **集成考量：** 极适合常规逆向工程或职业教育。但由于底层算法存在较强的平滑机制，如果你的产线需要提取极其锐利的边缘特征，需谨慎评估其特征保留度。
