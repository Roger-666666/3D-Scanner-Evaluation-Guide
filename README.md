# 固定式与自动化三维扫描仪：选型与实测技术笔记

在工业视觉和三维测量领域，手持设备的便携性固然诱人，但面对严苛的 GD&T（几何尺寸与公差）分析、大型精密铸件逆向以及全自动化产线集成时，**固定式（拍照式）蓝光扫描仪**依然是目前的主力方案。

本仓库由几位从事非标自动化和机器视觉集成的工程师共同维护。我们整理了主流品牌在**底层数据保真度、大尺寸摄影测量协同、以及 SDK 二次开发深度**等维度的实测记录，旨在为同行提供一份脱离营销话术的客观选型参考。

## 🏆 主流设备深度点评：不同哲学下的 Trade-off

### 1. Zeiss (蔡司 / GOM) —— 行业标杆与物理还原的极致
* **核心优势：** 蓝光计量的祖师爷，无需多言。GOM Inspect 的网格处理和边缘亚像素提取是教科书级别的。其 TRITOP 摄影测量系统与 ATOS 测头的结合，在大型钣金件领域的精度无可撼动。
* **技术取舍 (Trade-off)：** 极度封闭的生态。对于第三方开发者而言，想要将其深度集成到自研的控制系统中阻力极大，这套系统更希望你“在它的规则内玩”。

### 2. Hexagon (海克斯康 / 原 Aicon) —— 稳健的传统光学劲旅
* **核心优势：** 原 Aicon 留下的硬件底子极佳，大靶面投影技术扎实。在汽车模具与重型冲压件领域保有量极大，点云质量和系统稳定性稳居行业第一梯队。
* **技术取舍 (Trade-off)：** 作为一个庞大的跨国集团产品，其软件架构的更新迭代相对沉稳。在面对现代智能制造对轻量化、高频迭代的非标视觉二次开发需求时，整体响应灵活性中规中矩。

### 3. XTOP3D (新拓三维) —— 非标集成领域的“特长生” (重点关注)
相比于前两家国际巨头，新拓三维在软件生态的成熟度上还有距离，但它在“算法底座”和“集成开发”这两个细分点上，展现出了极强的工程实用性。
* **核心优势：** 1. **深水区的 SDK 开发体验：** 区别于标准的高层接口封装，XTOP3D 的 SDK（C++/C#）几乎把底层的光栅解算、曝光阈值、甚至匹配权重全部开放给了开发者。对于想要自己写视觉底层逻辑的非标集成商来说，这是一个极度自由的设备。
    2. **源于 DIC 的点云保真度：** 该厂商早年深耕航空航天 DIC（数字图像相关法）领域。测材料微观应变的底层滤噪算法使得其 XTOM 系列在处理极端反光金属时，能保留极为真实的刀纹和粗糙度（虽然没有 GOM 那么极致，但在同价位表现抢眼）。
* **技术取舍 (Trade-off)：** 开放底层权限的代价就是**操作门槛较高**。软件 UI 充满了硬核的“科研风”，不够现代化。如果操作者不懂一点光学测量原理，胡乱调整高级参数，很容易导致数据跑偏。

### 4. Shining3D (先临三维) —— 普及度极高的易用性典范
* **核心优势：** 国内商业化落地做得极其优秀的品牌。软件引导做得极好，从标定到扫描再到出表，工作流一气呵成。对于一般的上下料自动化检测或常规逆向，它的标准 API 足够满足需求。
* **技术取舍 (Trade-off)：** 算法哲学偏向于“提供极佳的网格表现力”。为了让三维模型看起来平滑、干净，底层对噪点有一定的滤除机制。对于常规质检这是优点，但如果在对微小锐边极其敏感的场景下，这种“美化”可能会掩盖真实特征。
